🚀 Project Overview

The Student Performance Analyzer is a simple yet powerful web application built using Python and Streamlit.
It helps analyze a student’s academic performance based on marks, attendance, and learning trends, and generates instant feedback and recommendations.

✨ Features
📊 Input student marks (Math, Science, English)
📅 Track attendance percentage
📈 Analyze performance trends (Improving / Declining / Stable)
🧠 Automatic performance evaluation
💡 Personalized improvement suggestions
🌐 Clean and simple web interface
🛠️ Tech Stack
🐍 Python
🌐 Streamlit
📊 Basic Data Analysis Logic
📦 Installation
1️⃣ Install Python

Download from:
https://www.python.org/downloads/

✔ Make sure to check “Add Python to PATH”

2️⃣ Install Streamlit
pip install streamlit
▶️ How to Run

Go to your project folder:

cd Desktop

Run the app:

streamlit run app.py

👉 If it doesn’t work:

python -m streamlit run app.py

🧠 How It Works
Takes student input (marks + attendance)
Calculates average score
Evaluates performance trend
Generates:
Strengths
Weak areas
Improvement suggestions
📊 Example Output
Average Score: 78
Attendance: 85%
Trend: Improving
💡 Analysis:
Good overall performance
Needs slight improvement in weak subjects
Attendance is satisfactory
