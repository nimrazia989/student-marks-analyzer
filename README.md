====================================================
        STUDENT PERFORMANCE ANALYZER
====================================================

A simple Python + Streamlit web application that
analyzes student academic performance based on
marks, attendance, and learning trends.

----------------------------------------------------
MADE BY
----------------------------------------------------
Nimra

----------------------------------------------------
PROJECT DESCRIPTION
----------------------------------------------------
This project helps to analyze student performance
by taking input of marks in different subjects,
attendance percentage, and performance trend.
It generates a simple report with suggestions
for improvement.

----------------------------------------------------
FEATURES
----------------------------------------------------
- Enter student name and subject marks
- Track attendance percentage
- Select performance trend (Improving / Declining / Stable)
- Automatic performance analysis
- Simple feedback and recommendations
- Easy-to-use interface

----------------------------------------------------
TECHNOLOGIES USED
----------------------------------------------------
- Python
- Streamlit

----------------------------------------------------
INSTALLATION
----------------------------------------------------

1. Install Python
Download from:
https://www.python.org/downloads/

IMPORTANT:
✔ Check "Add Python to PATH" during installation

2. Install Streamlit
Open CMD and run:
pip install streamlit

----------------------------------------------------
HOW TO RUN PROJECT
----------------------------------------------------

Step 1: Open Command Prompt
Step 2: Go to project folder (Desktop or where file is saved)

Example:
cd Desktop

Step 3: Run application

streamlit run app.py

OR if it doesn't work:

python -m streamlit run app.py

----------------------------------------------------
PROJECT STRUCTURE
----------------------------------------------------
app.py     -> Main Streamlit application
README.txt -> Project documentation

----------------------------------------------------
HOW IT WORKS
----------------------------------------------------
1. User enters student details
2. System calculates average marks
3. Attendance and trend are analyzed
4. Final report is generated
5. Suggestions are shown
